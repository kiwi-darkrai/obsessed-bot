## 🚀 INSTALLAZIONE RAPIDA

### 📱 TERMUX (ANDROID)
1. pkg update && pkg upgrade
2. pkg install nodejs git ffmpeg
3. git clone https://github.com/kiwi-darkrai/obsessed-bot
4. cd obsessed-bot
5. npm install
6. node index.js

### 💻 WINDOWS
1. Installa Node.js e Git.
2. Apri il terminale (CMD/PowerShell) e scrivi:
   git clone https://github.com/kiwi-darkrai/obsessed-bot
   cd obsessed-bot
   npm install
   node index.js

### 🍎 MACOS / 🐧 LINUX
1. git clone https://github.com/kiwi-darkrai/obsessed-bot
2. cd obsessed-bot
3. npm install
4. node index.js

## 🔗 COLLEGAMENTO
Il bot utilizza il sistema Pairing Code (senza QR):
1. All'avvio inserisci il tuo numero (es. 39333...).
2. Copia il codice di 8 cifre che appare nel terminale.
3. Su WhatsApp vai in: Dispositivi Collegati > Collega con numero di telefono.
4. Inserisci il codice e il bot sarà online.

## 📋 GUIDA AI COMANDI (Prefisso: .)

### 🛡️ MODERAZIONE (SOLO ADMIN)
- .apri : Apre il gruppo (tutti scrivono).
- .chiudi : Chiude il gruppo (solo admin scrivono).
- .muta : Muta un utente (i suoi messaggi verranno cancellati).
- .smuta : Togli il muto a un utente.
- .p : Promuovi un utente ad Admin.
- .r : Rimuovi i privilegi da Admin.
- .kick : Espelli un membro.
- .tagall : Tagga tutti visibilmente.
- .hidetag : Tagga tutti invisibilmente.
- .soloadmin : Il bot risponde solo agli admin.

### ☣️ OWNER (GODMODE)
- .nuke : Wipe totale (rinomina, chiude, tagga tutti ed espelle i membri).
- .niuk : Wipe estetico (rinomina, chiude e tagga tutti senza espellere).
- .recupera : Ripristina il nome e le impostazioni originali del gruppo.
- .bigtag [testo] [n] : Tagga tutti ripetutamente per "n" volte (max 100).
- .autoadmin : Ottieni i poteri da admin istantaneamente.
- .addowner : Aggiungi un nuovo proprietario al bot.
- .delowner : Rimuovi un proprietario dal bot.
- .block : Blocca un utente dal bot.
- .unblock : Sblocca un utente.
- .broadcast : Invia un messaggio a tutti i gruppi del bot.
- .shutdown : Spegne il bot dal terminale.

### 👤 INFO & SOCIAL
- .info : Dettagli utente (Dispositivo, Messaggi, Genere, IG).
- .setig : Imposta il tuo profilo Instagram nel database.
- .setgenere : Scegli Maschio o Femmina.
- .top : Classifica dei membri più attivi della giornata.
- .ping / .speed : Mostra la velocità di risposta del bot.

### 🎮 DIVERTIMENTO
- .ttt : Inizia una partita a Tris.
- .bandiera : Gioco di velocità con le emoji.
- .ship : Calcola l'amore tra due persone taggate.
- .sposa / .divorzia : Comandi social per il gruppo.
- .meme : Invia un meme casuale.

### 🛠️ SISTEMA
- .ds : Pulisce i file temporanei della sessione (Stile Purgatorio).
